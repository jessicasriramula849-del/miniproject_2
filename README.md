# miniproject_2
Ola Ride Business Insights - Miniproject2

Brief Summary of my project:

The rise of ride-sharing platforms has transformed urban mobility, offering convenience and affordability to millions of users. OLA, a leading ride-hailing service, generates vast amounts of data related to ride bookings, driver availability, fare calculations, and customer preferences. However, deriving actionable insights from this data remains a challenge. To enhance operational efficiency, improve customer satisfaction, and optimize business strategies, this project focuses on analyzing OLA’s ride-sharing data. By leveraging data analytics, visualization techniques, and interactive applications, my goal in this project is to extract meaningful insights that can drive data-informed decisions, in order to achieve this, I have created a streamlit application deployed it and also built a PowerBI dashboard in this project which are attached. 

For further information, you can read more in my attached project report here on github and view screenshots of my project.
